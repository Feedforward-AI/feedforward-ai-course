# VS Code: The Big Picture

Visual Studio Code (VS Code) is the application you're using right now. Think of it as a smart text editor designed for working with code and files.

## What Is It?

VS Code is like Microsoft Word, but for code. It's where you:
- Read and edit files
- Run commands in the terminal
- Talk to Claude

## Why Are We Using It?

For this course, VS Code gives you:
1. A **terminal** to talk to Claude
2. A **file explorer** to see documents and data
3. An **editor** to read and write files
4. Everything in one place

## Quick Start

1. **See your files**: Look at the left sidebar (or press `Cmd+Shift+E` / `Ctrl+Shift+E`)
2. **Open the terminal**: Press `` Ctrl+` `` (backtick, below Escape)
3. **Start Claude**: Type `claude` in the terminal and press Enter
